旅程明细

图 6 · 各站点进入 / 转化 / 停留 / 工单明细，含分大区停留列。
口径说明 —— 「期间进入」= 所选日期范围内首次到达该站的用户数；「当前停留」= 截至最近刷新时点仍停留在该站、未流转到下一站的用户数（快照口径，仅受大区筛选影响）；「合同激活」为旅程终点，不计停留。「在办工单」= 特定问题场景下已触发系统工单、由后台处理中的客户数（快照口径）；「反馈后回流率」= 工单反馈后 24h 内回到主旅程并推进的比例；「协助推进观测」三口径以 24h 内推进到下一站为准。 本页为产品看板原型，数据由前端按日期种子模拟生成（每天刷新出新数值）；接入生产时替换 loadData() 为数仓 / API 查询即可，页面结构与交互不变。
